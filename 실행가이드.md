# 핀볼 축구 게임 - 실행 가이드

## 버전 1.0 (2026-07-15)

---

## 1. 시스템 구성

- **Server**: Node.js + WebSocket (`ws` 라이브러리) — 방 관리 및 게임 상태 동기화
- **Client**: HTML5 Canvas — 순수 JavaScript, 의존성 없음
- **통신**: WebSocket 실시간 양방향 통신

## 2. 로컬 실행 방법

### 2.1 서버 실행
```bash
cd server
npm install
npm start
```
기본 포트: **3000**

### 2.2 접속
브라우저에서 `http://localhost:3000` 접속

## 3. 클라우드 배포 방법

### 3.1 Render.com (추천, 무료)

#### 방법 A: Blueprint 자동 배포 (권장)
1. [Render.com](https://render.com) 계정 생성 (GitHub 연동)
2. 대시보드 → **New +** → **Blueprint**
3. GitHub 저장소 연결 → 자동 감지 및 배포

#### 방법 B: Web Service 수동 설정
1. [Render.com](https://render.com) 계정 생성 (GitHub 연동)
2. 대시보드 → **New +** → **Web Service**
3. GitHub 저장소 연결
4. 설정:
   - **Root Directory**: *(비워둠)*
   - **Build Command**: `cd server && npm install`
   - **Start Command**: `node server/server.js`
   - **Instance Type**: Free
5. 배포 완료 후 `https://[app-name].onrender.com` 접속

### 3.2 Railway.app (무료)
1. [Railway.app](https://railway.app) 계정 생성
2. **New Project** → **Deploy from GitHub**
3. 저장소 연결 후 서버 자동 배포
4. 설정 변경 없음 (자동 감지)

### 3.3 Fly.io
```bash
fly launch
fly deploy
```

## 5. 게임玩法

### 5.1 방 만들기
1. **방 만들기** 버튼 클릭
2. 닉네임 입력
3. **방 생성** → 2자리 코드 생성 (예: `47`)
4. 코드를 친구들에게 공유

### 5.2 방 참여
1. **방 참여하기** 버튼 클릭
2. 닉네임 입력
3. 2자리 코드 입력
4. **참여하기**

### 5.3 게임 플레이
- **마우스 이동**: 골키퍼 위치 조정 (아레나 가장자리를 따라 이동)
- **마우스 클릭/터치**: 골키퍼가 튀어나가 공을 막음 (Spring)
- **목표**: 내 골대에 공이 들어가지 않도록 막기 (실점 최소화)

### 5.4 게임 규칙
- 최대 8명 동시 플레이
- 각 플레이어는 아레나 주변에 위치한 골대를 방어
- 핀볼 스타일: 공이 범퍼와 벽에 튕겨 다님
- 공이 내 골대에 들어가면 실점
- 게임 종료 후 실점이 가장 적은 플레이어 승리

## 7. 포트 변경
환경 변수 `PORT` 설정으로 변경 가능:
```bash
$env:PORT=8080; npm start   # Windows PowerShell
PORT=8080 npm start          # Mac/Linux
```
